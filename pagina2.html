Introducción de la estructura de una página web considerando (HTML, CSS, JS)

La estructura de presentación de una página web con HTML, CSS y JS es fundamental para la creación de páginas web modernas y atractivas para los usuarios. HTML permite definir la estructura de la página, CSS permite darle estilo y diseño, mientras que JS permite hacerla interactiva y dinámica.

La combinación de estos tres lenguajes permite a los desarrolladores web crear páginas web con una gran variedad de características y funcionalidades, desde simples páginas informativas hasta complejas aplicaciones web.

Es importante entender cómo funciona cada uno de estos lenguajes y cómo se relacionan entre sí para poder crear páginas web eficientes y efectivas. La estructura de presentación de una página web es una herramienta esencial para los desarrolladores web, ya que les permite organizar y diseñar la página de manera clara y coherente.

  1. HTML

    • Inicio del documento: se utiliza la etiqueta <!DOCTYPE> para indicar el tipo de documento.
    • Encabezado: se define con la etiqueta <head> y contiene información sobre la página, como el título y las hojas de estilo CSS.
    • Cuerpo: se define con la etiqueta <body> y contiene el contenido de la página, como textos, imágenes, videos, formularios, entre otros.

  2. CSS

    • Selector: indica los elementos a los que se les aplicará el estilo.
    • Propiedades: se definen los valores que se van a aplicar a los elementos seleccionados, como el color, la fuente, el tamaño, la posición, entre otros.
    • Agrupación: se pueden agrupar varios selectores con las mismas propiedades para simplificar el código.

  3. JavaScript

    • Evento: indica la acción que activa el script, como un clic, una carga o un cambio.
    • Función: se define el código que se va a ejecutar cuando se activa el evento, como mostrar un mensaje, validar un formulario o realizar un cálculo.
    • Agrupación: se pueden agrupar varios selectores con las mismas propiedades para simplificar el código.
