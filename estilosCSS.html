Principales estilos para CSS y la definición de elementos

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo utilizado para describir cómo se presenta un documento HTML. CSS se utiliza para definir la apariencia visual de los elementos HTML en una página web.

Los principales estilos de CSS incluyen:

Selector de Elemento: Es el estilo predeterminado de CSS y se aplica a todos los elementos en una página web. Por ejemplo: 
              body {
                background-color: #FFFFFF;
                font-family: Arial, sans-serif;
              }
Selector de ID: Se utiliza para aplicar estilos a elementos HTML que tienen un atributo de identificación único. Por ejemplo: 
                #header {
                  background-color: #333333;
                  color: #FFFFFF;
                  font-size: 24px;
                }
Selector de Clase: Se utiliza para aplicar estilos a elementos HTML que tienen una clase específica. Por ejemplo: 
                .button {
                  background-color: #007FFF;
                  color: #FFFFFF;
                  padding: 10px 20px;
                }
Selector de Atributo: Se utiliza para aplicar estilos a elementos HTML que tienen un atributo específico. Por ejemplo: 
                input[type="text"] {
                  input: #007FFF;
                  border: 1px solid #CCCCCC;
                  padding: 5px;
                }
